MVV [My Vacay Valet]

Project Description

My Vacay Valet (MVV) is a grocery delivery service for families and individuals who are looking to skip the hectic grocery stores and go straight to relaxing. Customers shop orders via an online storefront running Miva Merchnat. The payments are processed through Miva's secure system. The information is then sent to the employee web application for employees to shop for the groceries, deliver groceries, and to manage inventory.

At this point in time MVV is still in the preliminary stages of development and not ready for production.

Development Enviroment

Java Server Pages (JSPs), Tomcat Server, MySQL

Project Details

  • Google Maps API for delivery scheduling and tracking
  • Location specific grocery information (Web Scrapper using nightmare.js)
  • Inventory Mangement
  • Google Data Studio reporting system for live information
